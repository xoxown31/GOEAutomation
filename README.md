# GOE Automation - GOE 메신저 자동화

GOE 메신저 자동화를 위해 만들어 본 파이썬 코드입니다.\
아직 구현에만 초점이 맞춰져 여러 환경에 대한 잡다한 버그가 있을 수 있으며,\
리팩토링이 되지 않아 전부 함수로 선언되어 있습니다.
